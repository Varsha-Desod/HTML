Form:- From are allow user to input data and submit it.
There are two method of form
1. GET (which is not secure because when user give input like username and password in the address field the usernameand password are shown thats why its not secure).
2. POST (which is secure because when user give input in address field the given data are not shown thats why its not secure).
Some Important attributes:-
1. Action attribute :- used to get path of that file which holds the database or from where send sent or store the data in the form.
2. Required attribute :- used to when user are forgot or not field any input and submit the form this attribute can't submitted the form and show the red color error that "THIS IS IMPORTANT FIELD PLEASE FILL IT ".
3. Placeholder attribute :- used to show hint in side the placeholder.
4. maxlength :- used to control the mximum length of a character and number.
5. rows :- used to provide limited rows in a address field.
6. cols :- used to provide limited column in a address field.
7. fieldset Tag :- is used to create a box or a line with text using legend tag.
8. select tag :- used to create drop down list field.
9. option tag :- used to create drop down list.
10. textarea tag :- used to create a area for address
